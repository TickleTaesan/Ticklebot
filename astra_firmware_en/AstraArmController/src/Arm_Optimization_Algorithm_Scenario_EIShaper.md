# EIShaper (입력 셰이핑) 알고리즘 시나리오

1. **명령 입력**
   - 새로운 위치 명령(배열)이 입력된다.

2. **버퍼에 현재 위치 저장**
   - 현재 위치 명령을 순환 버퍼에 저장한다.

3. **지연 위치 추출**
   - 버퍼에서 t2, t3만큼 지연된 과거 위치 명령을 추출한다.

4. **보간(필요시)**
   - t2, t3가 정수가 아니면, 인접한 두 버퍼 값으로 선형 보간하여 정확한 지연값을 계산한다.

5. **가중합 계산**
   - 현재 위치, t2 지연 위치, t3 지연 위치를 각각의 가중치(A1, A2, A3)로 곱해 합산한다.

6. **셰이핑된 명령 출력**
   - 최종적으로 진동 억제 효과가 적용된 위치 명령을 다음 단계(트래젝토리 플래너 등)로 전달한다.

---

이 시나리오는 AstraArmController의 EIShaper 알고리즘이 진동 억제를 위해 입력 명령을 실시간으로 가공하는 과정을 단계별로 설명합니다.
